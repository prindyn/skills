# Pre-Launch Checklist — [Project Name]

> Run through this *every page*, *every device*, before hitting publish. The cost of catching things now is small; the cost of a broken launch is high.

## Content

- [ ] All placeholder text (`Lorem ipsum`, `Coming soon`, `[YOUR NAME HERE]`) removed
- [ ] Proofread every page for typos and grammar (read aloud, or use a tool)
- [ ] All names, prices, dates, contact info verified accurate
- [ ] Every link clicked at least once — internal and external — to confirm no 404s
- [ ] Every image has descriptive alt text (accessibility + SEO)
- [ ] No copyrighted images / fonts used without license

## Design & layout

- [ ] Visual hierarchy clear on every page (the most important thing is the most prominent thing)
- [ ] Consistent fonts, colors, button styles across all pages
- [ ] Whitespace adequate — no crowded sections
- [ ] Mobile layout reviewed page-by-page (don't trust "responsive" — actually look)
- [ ] Tablet layout reviewed (often the worst of both worlds)
- [ ] Color contrast passes WCAG AA (verify with a contrast tool)

## Functionality

- [ ] Every form submits successfully and you receive the email/notification
- [ ] Form validation works (try empty fields, bad email format)
- [ ] Every CTA button goes where it should
- [ ] Navigation menu works on mobile (hamburger opens/closes, links work)
- [ ] Search works (if applicable)
- [ ] E-commerce: full purchase flow tested with a real card / sandbox

## Performance

- [ ] Images compressed (use WebP where possible; aim for <200KB per image)
- [ ] Page load tested — homepage should be under 3 seconds on a normal connection
- [ ] No autoplay audio (and ideally no autoplay video with sound)
- [ ] Hero video (if any) has a static fallback for slow connections

## SEO & metadata

- [ ] Every page has a unique title tag (50–60 chars)
- [ ] Every page has a unique meta description (150–160 chars)
- [ ] OpenGraph image set (so the site looks right when shared on social)
- [ ] Favicon uploaded
- [ ] Sitemap.xml generated and submitted to Google Search Console
- [ ] robots.txt configured (don't accidentally block search engines)

## Analytics & tracking

- [ ] Google Analytics (or alternative) installed and firing
- [ ] Conversion events configured for the primary CTA(s)
- [ ] Test events firing in the analytics dashboard

## Legal & compliance

- [ ] Privacy policy live and linked in footer
- [ ] Terms of service live (if applicable)
- [ ] Cookie banner (if required by jurisdiction or analytics setup)
- [ ] Email collection complies with applicable laws (CAN-SPAM, GDPR, etc.)

## Domain, hosting, security

- [ ] Custom domain connected and resolving (`yourdomain.com` AND `www.yourdomain.com`)
- [ ] HTTPS / SSL certificate active (the lock icon shows in the browser)
- [ ] Old / staging URLs no longer publicly accessible
- [ ] 301 redirects set up if migrating from an old site
- [ ] Backup taken of the final pre-launch state

## Cross-browser

Spot-check the homepage and one key conversion flow in:
- [ ] Chrome (desktop + mobile)
- [ ] Safari (desktop + iOS)
- [ ] Firefox
- [ ] Edge

## Final sign-off

- [ ] Owner has reviewed every page on a phone
- [ ] Owner has approved going live
- [ ] Launch announcement drafted (email, social, etc.) and ready to send
